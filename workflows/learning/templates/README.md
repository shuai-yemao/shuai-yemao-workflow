# 学习系统模板使用说明

## 模板列表

### 1. 每日自评模板 (`daily-review.md`)

**用途：** 每天学习后进行自我评估

**使用时机：** 每天学习结束后

**使用方法：**
1. 复制模板到Obsidian的`06_日志`文件夹
2. 重命名为当天日期，如`2024-01-15 学习自评.md`
3. 按照模板填写内容
4. 重点完成"一句话解释测试"

**核心步骤：**
```
1. 列出今天学了什么
2. 每个内容用一句话解释
3. 标记理解程度（✓/△/✗）
4. 问AI验证你的理解
5. 记录修正后的理解
6. 规划明天的复习
```

---

### 2. 概念验证模板 (`concept-verification.md`)

**用途：** 用费曼技巧验证对单个概念的理解

**使用时机：** 学习新概念后，或复习时验证理解

**使用方法：**
1. 复制模板到Obsidian的`03_概念`文件夹
2. 重命名为概念名称，如`简单工厂.md`
3. 按照模板逐步填写

**核心步骤：**
```
1. 用自己的话解释（假设教10岁小孩）
2. 寻找生活中的类比
3. 问AI验证你的解释
4. 根据反馈修正理解
5. 通过应用来验证
6. 记录验证结果
```

---

## 每日学习流程

### 早晨（5分钟）
1. 打开Obsidian
2. 查看今天的自评模板
3. 确定今天要学什么

### 学习中（按需）
1. 学习新内容
2. 每学完一个概念，用概念验证模板
3. 遇到问题，问AI老师

### 晚上（10分钟）
1. 打开今天的自评模板
2. 完成"一句话解释测试"
3. 问AI验证理解
4. 记录疑问和反思
5. 规划明天的学习

## 如何使用

### 方法1：复制到Obsidian
1. 打开Obsidian知识库
2. 创建`模板`文件夹
3. 复制模板文件到该文件夹
4. 在Obsidian中设置模板文件夹路径

### 方法2：直接在Obsidian中创建
1. 打开Obsidian
2. 新建笔记
3. 粘贴模板内容
4. 按照模板填写
